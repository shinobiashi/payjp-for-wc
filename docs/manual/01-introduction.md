# 第1章 はじめに

## 1-1. このプラグインでできること

**PAY.JP for WooCommerce** は、WooCommerce のネットショップに次の決済手段を追加するプラグインです。

### クレジットカード決済

- お客様は購入画面（チェックアウト画面）から離れることなく、その場でカード情報を入力して支払いできます。
- Visa / Mastercard / JCB など、PAY.JP が対応する主要ブランドが利用できます。
- お客様がカード情報を保存しておき、2 回目以降の購入で入力を省略することもできます。

### PayPay 決済

- お客様はチェックアウト画面から PayPay で支払いできます。
- スマートフォンの PayPay アプリ、または QR コードで支払いが完了します。

### そのほかの特長

- **安全です** — お客様のカード番号は PAY.JP が提供する安全な入力フォームで直接処理されます。あなたのサイトやサーバーにカード番号が保存・通過することはありません（PCI DSS というカード業界のセキュリティ基準に準拠した方式です）。
- **返金も管理画面から** — 全額返金・一部返金を WooCommerce の注文画面から行えます。PAY.JP のダッシュボードを開く必要はありません。
- **定期購入に対応** — WooCommerce Subscriptions（別売りの拡張）と組み合わせると、毎月の自動課金などの定期決済ができます。
- **新旧どちらのチェックアウト画面にも対応** — WooCommerce の新しいブロック型チェックアウトと、従来型のチェックアウトのどちらでも動作します。

## 1-2. 必要な環境

お使いのサイトが次の条件を満たしているか確認してください。分からない場合は、サイトの制作を依頼した会社やサーバー会社にお問い合わせください。

| 項目 | 必要なバージョン |
| ------ | ---------------- |
| WordPress | 6.9 以上 |
| WooCommerce | 9.0 以上 |
| PHP（サーバーのプログラム実行環境） | 8.3 以上 |
| PAY.JP アカウント | 無料で作成できます（第2章で説明します） |

> [!NOTE]
> WordPress と WooCommerce のバージョンは、WordPress 管理画面の「ダッシュボード > 更新」や「プラグイン」画面で確認できます。PHP のバージョンは「ツール > サイトヘルス > 情報 > サーバー」で確認できます。

## 1-3. 費用について

- プラグイン自体は**無料**です。
- PAY.JP のアカウント作成・月額利用も基本プランは**無料**です。
- 決済ごとに PAY.JP の **決済手数料（料率）** がかかります。料率はプランやカードブランドによって異なります。最新の料率は [PAY.JP の料金ページ](https://pay.jp/) でご確認ください。

> [!IMPORTANT]
> 本プラグインの利用者は、本番申請の完了後に**特別料率の申請フォームから申請することで、特別料率が適用されます**。
> この申請を行わないと通常料率のままになります。詳しくは「[第6章 本番利用の申請と本番切り替え](06-go-live.md)」をご覧ください。
