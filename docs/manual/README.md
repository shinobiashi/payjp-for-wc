# PAY.JP for WooCommerce ユーザーマニュアル

このマニュアルは、WooCommerce で作ったネットショップに「PAY.JP for WooCommerce」プラグインを導入し、クレジットカード決済・PayPay 決済を受け付けられるようにするための手順書です。

**IT にあまり詳しくない方でも順番に読み進めれば設定できる**ように書いています。上から順に進めてください。

---

## 導入の全体の流れ

```
① PAY.JP のアカウントを作る（無料）
        ↓
② プラグインをインストールする
        ↓
③ テスト用の設定をする（テストモード）
        ↓
④ テスト決済で動作を確認する（お金は動きません）
        ↓
⑤ PAY.JP に本番利用を申請する（審査があります）
        ↓
⑥ 審査通過後、本番用の設定に切り替える
        ↓
⑦ ★特別料率の申請フォームから申請する ← 忘れると通常料率のままです！
        ↓
⑧ 販売開始・日々の運用
```

> [!IMPORTANT]
> **⑦ の「特別料率の申請」は自動では行われません。**
> 本番申請が完了しただけでは特別料率は適用されず、**指定のフォームからの申請が別途必要**です。
> 詳しくは「[第6章 本番利用の申請と本番切り替え](06-go-live.md)」を必ずお読みください。

## 用語について

このマニュアルでは、はじめての方向けに次の言葉を使います。

- **PAY.JP（ペイドットジェーピー）** … クレジットカード決済などを代行してくれる日本の決済サービスです。お客様が入力したカード情報は PAY.JP が安全に処理するため、あなたのサイトにカード番号が保存されることはありません。
- **ダッシュボード** … PAY.JP にログインすると表示される管理画面のことです。
- **API キー** … あなたのサイトと PAY.JP をつなぐための「合言葉」のような文字列です。テスト用と本番用の 2 種類があります。
- **テストモード** … 実際にお金が動かない練習モードです。設定やテストはまずこのモードで行います。
